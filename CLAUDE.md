## Commands

```bash
npm run dev            # local dev server (localhost only)
npm run dev:phone      # dev server exposed on LAN — use the Network URL on phone
npm run build          # production build
npm run preview:phone  # serve production build on LAN for phone testing
npm run lint           # oxlint
```

All installs require `--cache /Users/suresh/Apps/npm-cache` due to a root-owned npm cache issue on this machine:
```bash
npm install <pkg> --cache /Users/suresh/Apps/npm-cache
```

## Architecture

Single-page React PWA built with Vite + `vite-plugin-pwa`. No router — everything lives on one page.

**Layout** (`src/App.jsx`): fixed Header (56px) + flex body + fixed Footer (60px). The body splits into two columns:
- Left 50%: `BibleText` — full body height, scrollable
- Right 50%: `SongBook` (top half) stacked over `Dictionary` (bottom half)

**PWA install** (`src/hooks/useInstallPrompt.js`): listens for `beforeinstallprompt` (Android Chrome). iOS is detected via `navigator.userAgent` — the Footer shows a manual Share→Add to Home Screen instruction instead.

**Service worker** is auto-generated by `vite-plugin-pwa` (Workbox, `autoUpdate` mode). No hand-written SW.

**Icons**: `public/icons/icon-192.png` and `icon-512.png` are referenced in the Vite PWA manifest config (`vite.config.js`) and in `index.html` for the iOS apple-touch-icon.

## Phone Testing

Phone and computer must be on the same WiFi. Run `npm run dev:phone`, then type the `Network:` URL printed in the terminal into the phone browser. For Android install testing use Chrome; for iOS use Safari.
